Le 24-07-2016

NuTyX 8.2-beta-2 disponible avec cards 2.1.84

Ces deux dernières semaines ont été très prolifique au niveau de la future 8.2.

Deux nouvelles ISO 64bits sont proposées pour cette nouvelle version. La première (1.0G) est une version de kde5 installable kf5 5.24.0 et la deuxième (221M) est une ISO standard MINI.

Pensez à vérifier leur somme MD5 avant de les transférer sur clé USB, CD-RW ou DVD-RW

L'installateur

Toutes les actions sont désormées enrégistrées dans le fichier /var/log/install.

Une fois NuTyX installée sur votre disque dur, l'installateur vous demandera de créer un nouvel utilisateur. Modifiez les valeurs de défaut user et Description sinon cela ne fonctionnera pas.

Les nouveautées de NuTyX

La commande nu est supprimée et remplacée par la commande setup-nutyx avec l'option -cu. Pour ne pas perturber les utilisateurs, un alias est disponible dans le fichier de configuration .bashrc: 

alias nu='sudo setup-nutyx -cu'

Le compte root n'est plus activé par défaut. Vous vous connectez désormais directement avec votre compte personnel créé lors de l'installation. Pour activer le compte root, il suffit de passer par la commande setup-nutyx ou nu, choisissez root comme utilisateur et spécifiez un nouveau mot de passe.

Tous les paquets ont été recompilés avec la nouvelle version du compilateur gcc 6.1 pour assurer une stabilitée maximum.

Le fichier /etc/inittab est désormais responsable pour le lancement de lxdm. Avec cette methode. celui-ci se relance automatiquement si il y a une erreur à la première tentative de lancement.

Toutes les collections habituelles sont disponibles. Les ports des collections lxde et lxde-extra sont désormais intégrés dans les collections standards gui et gui-extra.

Vous pouvez consulter le changelog pour connaître les mise à jour sur le système core

Cards 2.1.84

Deux nouveaux scripts sont ajoutés build-port et build-collection, ils permettent donc de construire respectivement un ensemble de ports et un ensemble de collection. Vous trouverez plus d'info sur le projet de cards

cards permet désormais d'effacer une collection complète, une suite de paquets ou un groupe de paquets. La collection d'un paquet installé est également stocké dans la base de données.

cards stocke désormais le nom de la collection, du groupe et de la famille du paquet. Avec ces informations supplémentaires, il est donc possible d'obtenir des informations plus filtrées à l'affichage de la sortie de la commande cards list par exemple.

Le script d'installation

Le script bash install-nutyx-8.2-beta-2 permet d'installer NuTyX depuis votre NuTyX 8.1 stable ou depuis n'importe quel autre distribution.

Le site nutyx.org

Hazel Russman a relu les pages anglaises du site. La documentation anglaise est désormais beaucoup plus proche de la version française.

Une nouvelle page explique comment utiliser le script enter-nutyx (sera renommé en build-collection). Cet article est surtout utile pour les packagers.
