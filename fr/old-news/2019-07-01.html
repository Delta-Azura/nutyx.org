Dernière nouvelle 01-07-2019

NuTyX 11.1 disponible avec cards 2.4.98

J'ai le grand plaisir de vous annoncer la sortie de la nouvelle NuTyX version 11.1.

Cette nouvelle version contient plus de 1000 mise à jour.

Une version 32 bits de NuTyX 11.1 est également disponible.

La base de NuTyX est fournie avec le nouveau kernel LTS 4.19.56 (4.9.183 pour la version 32 bits) et le tout nouveau kernel 5.1.15 (uniquement en 64 bits).

Le compilateur gnu gcc est passé en version 9.1.0

Le serveur graphique xorg-server version 1.20.5, la librairie Mesa 3D en 18.3.6, gtk3 3.24.9 et qt 5.12.4 sont également dans leurs dernières versions

Les interpreteurs python sont en version 3.7.3 et 2.7.16.

L'environnement de bureau MATE est également mis à jour en version 1.22.1, dernière version disponible.

L'environnement de bureau KDE est disponible en Plasma 5.16.2, Framework 5.59.0 et les applications en 19.04.2

Les navigateurs disponibles sont entre autre: firefox 67.0.4 et chromium 74.0.3729.169 (compilé par l'équipe de ArchLinux et uniquement en 64 bits)

De nombreuses applications on été mise à jour également comme thunderbird 60.7.2, Scribus 1.5.5, libreoffice 6.2.4.2, gimp 2.10.12, etc.

4 nouvelles ISOs sont disponible.

  • en 64 bits:

    Une ISO NuTyX de base de 384 Moctets en version "rolling".

    Une ISO de 5.3 Goctets est une NuTyX de base version "fixed" qui contient un dépot complet des paquets binaires de la NuTyX 11.1. Avec cette ISO, vous pouvez installer une NuTyX 11.1 complète sans avoir recours à une connection internet. Pour installer des paquets depuis cette ISO, une fois votre système de base installé sur votre disque dur, il faut monter la clé USB sur le dossier /mnt.

    Supposons que votre clé est trouvé sous /dev/sdc1, utilisez la commande: 

    sudo mount /dev/sdc1 /mnt
    Adaptez en conséquence.

CARDS

CARDS gère désormais les paquets obsètes. Un paquet est obsolète lorqu'il n'est plus disponible sur le serveur distant. Un tel paquet est supprimé automatiquement.

Mise à jour

La NuTyX 11.1 est une mise à jour mineure, une mise à jour est donc possible.

Il est impératif de mettre à jour le paquet cards en premier.

Veuillez utilisez ces quelques commandes pour effectuer une mise à jour sans risque: 

check
On met donc cards à jour: 
up cards
Ensuite on peut terminer la mise à jour: 
sudo cards upgrade
