Le 26-09-2016

NuTyX 8.2-rc2 disponible avec cards 2.1.93

L'équipe de NuTyX est très fière de vous proposer la nouvelle version 8.2-rc2.

Une nouvelle ISO 64bits MINI est proposée pour cette nouvelle version. Sa taille fait 217M.

Pensez à vérifier leur somme MD5 avant de les transférer sur clé USB, CD-RW ou DVD-RW.

Nouveautées pour l'utilisateur

L'installateur va vous demander si vous souhaitez activez les mise à jour automatiques.

Si disponible, l'outil de configuration trouvera automatiquement les points d'accès WIFI disponible et vous demandera quel ESSID vous souhaitez vous connecter. Vous pouvez alors définir votre point d'accès favori.

Quelques bugs découverts dans la RC1 ont été corrigé dans la RC2.

La documentation a été mise à jour.

Le nouveau site web sera en ligne en même temps que la sortie de la NuTyX 8.2 stable. Le site sera généré dynamiquement par cards.

Le script bash install-nutyx-8.2-rc2 permet d'installer NuTyX depuis votre NuTyX 8.1 stable ou depuis n'importe quel autre distribution.

La suite

Si aucun problème majeur n'est découvert, la prochaine version sera la version stable 8.2 en début d'octobre.
