Dernière nouvelle 11-01-2017

NuTyX 8.2.93 disponible avec cards 2.1.105

L'équipe de NuTyX est très fière de vous proposer une nouvelle version de dévelopement de NuTyX 8.2.93.

Plus de 1500 commits depuis la version 8.2

Des nouvelles isos sont disponible en 32 et 64 bits. Leurs tailles sont respectivement 219 MB et 224 MB.

Les interfaces graphiques disponible sont: kde5, gnome, mate, xfce4, lxde, flwm, twm, jwm, blackbox, openbox, ratpoison.

Troisième version de dévelopement de la prochaine version stable NuTyX 8.3 prévu pour début mars au plus tard.

L'installateur

La post-configuration du wifi a été amélioré. Le bug de l'ESSID corrigé. Un nom composé est désormais possible.

L'initrd n'étant plus necessaire, celle-ci n'est plus copiée dans l'installation. La ligne initrd est commentée dans le fichier grub.cfg.

Cards 2.1.105

Cards affiche plus de résultats lors d'une recherche de paquets. Si le paquet trouvé est déjà installé, celui-ci s'affichera en vert dans le résultat de la recherche. Désormais, celle-ci s'effectue également dans les alias et les groupes.

Python 3 est désormais la version par défaut

La version de Python est passée en version 3.6.0.

L'occasion pour l'équipe de NuTyX de renommer les ports python en python2 et python3 en python.

Tous les paquets ayant comme dépendances Python (version 2 et 3) ont été recompilés.

La version Qt par défaut est la version 5.8-rc

Qt est désormais splité en suivant les recommandations de l'équipe de developement.

KDE 5 nécessite seulement une partie de Qt.

Tous les paquets ayant comme dépendances un des modules de Qt (version 5) ou Qt4 ont été recompilés.

Autres Modification de noms de paquets

xorg-mesa s'appelle mesa

fltk1 s'appelle fltk
