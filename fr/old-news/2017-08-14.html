Dernière nouvelle 14-08-2017

NuTyX 9.0.93 disponible avec cards 2.3.93

L'équipe de NuTyX est très fière de vous proposer la version NuTyX 9.0.93

NuTyX 9.0.93 propose le kernel 4.12.7, kernel lts 4.9.43, glibc 2.26, gcc 7.2.0, binutils 2.29, python 3.6.2, xorg-server 1.19.3, qt 5.9.1, mate 1.18.0, xfce4 4.12.3, firefox 55.0.1 et beaucoup d'autres mise à jour.

Plus de 4000 commits depuis la version 9.0

Une nouvelle iso 64 bits est disponible. La taille est de 246 MB sur http://downloads.nutyx.org.

Puisque toutes les collections sont en cours de recompilations, aucune interfaces n'est actuellement disponible

Mise à jour majeure

Cette nouvelle version est le résultat de plus de 4 mois de dévelopement. Une mise à jour la 9.0 n'est pas possible car risquée, l'équipe de NuTyX recommande de faire une nouvelle installation.

Nouveautées dans les ISOs

Cette iso peut fonctionner en mode live, cad que si vous avez assez de mémoire, il n'est pas necessaire de l'installer, vous pouvez tester cette nouvelle NuTyX entièrement en RAM

La clé USB peut donc être déconnectée et ainsi libérer le port USB. Utile dans le cas de l'installation sur tablette ne disposant que d'un seul port USB

Les composants principaux de la chaîne de compilation sont mis à jour.

Les composants principaux de la chaîne d'outils cad en-têtes du noyau, compilateur, editeur de liens, assembleur et librairie C sont désormais à jour, les versions respectives de chaque composant se trouvent dans la liste ci-dessous.

Le nombre de collections officielles passe de 13 à 11 collections:

  • base
  • cli, cli-extra
  • gui, gui-extra
  • xfce4, xfce4-extra
  • mate, mate-extra
  • gnome, gnome-extra

Renommage et Splitting

Le renommage de gstreamer1 en gstreamer et d'autres paquets qui en dépendent entraîne plus de travail que prévu. L'équipe de NuTyX recompile toutes les collections.

  • webkit2gtk2 -> webkit
  • gstreamer1 -> gstreamer
  • ancienne version de gstreamer supprimée
  • qt4 supprimée

Cards 2.3.93

Beaucoup d'améliorations ont été faites sur la construction des paquets

En hommage à Pierre qui nous a quitté il y a quelques semaines, Cards affiche désormais le nom de tous les contributeurs. Son nom ne sera jamais oublié au sein de l'équipe Cards recherche également dans le nom du packager. Cards peut désormais installer un paquet ou une collection ou un fichier archive ou un mélange des trois. Il peut désormais supprimer une famille de paquets cad un ensemble de paquets dont le début du nom est commun. Si le paquet trouvé est déjà installé, celui-ci s'affichera en vert dans le résultat de la recherche. Désormais, celle-ci s'effectue également dans les alias et les groupes. Des nouvelles post-installations génériques sont désormais intégrées dans cards. Un site reprenant la documentation complète de cards a été créé: http://cards.nutyx.org
