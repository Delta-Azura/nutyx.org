Latest news 2016-12-18

NuTyX 8.2.92 available with cards 2.1.103

The NuTyX team is please to annonce the development release 8.2.92 of NuTyX.

About 1400 commits since version 8.2

New Isos are available. The size is 224M for the 64 bits version and the 32 bits version.

Following graphical interfaces are availables: kde5, mate, xfce4, lxde, flwm, twm, jwm, blackbox, openbox, ratpoison.

Already the second development version of the next stable NuTyX 8.3 planned for beginning of mars 2017. With the development branch, NuTyX propose now a rolling release version or RR. The life time of this RR is link to the life time of the current development branch of NuTyX.

The new kernel-lts

With this version of NuTyX, they is no need to specify an initrd anymore, all commons filesystems are now compiled into the kernel.

The kernel-lts is moving on to version 4.9.0, This LTS version is actualy the latested available version means for the moment they are no "standard" kernel available.

slim

The slim login manager is back. The big advantages of this LM, it's very light and can detect automaticaly all the available graphical interfaces
